Configuración inicial de un proyecto backend con nodejs y framerwork express

Quitar restricción de paquetes npm de windows (abrir powershell cómo administrador)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

descargar node
https://nodejs.org/es/

iniciar proyecto backend
npm init 
npm init -y

limpiar consola
(clear)

librerías para iniciar proyectos
npm i express cors bcrypt jsonwebtoken multer mysql2

en el package.json dentro de scripts
    "start": "node server.js",
    "dev": "nodemon server.js"
otra forma
    "start": "node index.js",
    "dev": "nodemon index.js"
estos comandos sirven para inicializar o correr el proyecto.


cuando se crea el archivo server.js, se cambia en el package.json de indez a server 
"main": "index.js", -> "main": "server.js",
